# NexusServe Web App (PWA)

NexusServe is a professional web application concept for a two-sided service marketplace where customers request services and the platform auto-contracts the best available provider.

## Core capabilities
- Customer request intake for multiple categories (plumbing, trash disposal, internet setup, electrical)
- Automated provider matching using service type, provider quality score, and priority-weighted ETA
- Transparent fee model from both sides:
  - Customer platform fee: 4.5%
  - Provider commission: 8.0%
- Live marketplace metrics simulation
- Installable Progressive Web App setup (manifest + service worker)

## Run locally
Open `index.html` in a browser, or serve the directory with a static server.
